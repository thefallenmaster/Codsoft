# CodSoft Python Projects

This repository contains Python mini projects created for the CodSoft internship program. Each task focuses on solving a practical problem using core Python concepts such as user input handling, conditional logic, loops, functions, and simple project structuring.

## Task 1: To-Do List

Build a task manager application that allows users to add, update, delete, and view daily tasks. The goal of this project is to practice list handling, menu-driven programming, and basic CRUD operations in Python.
