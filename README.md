# CodSoft Python Projects

This repository contains Python mini projects created for the CodSoft internship program. Each task focuses on solving a practical problem using core Python concepts such as user input handling, conditional logic, loops, functions, and simple project structuring.

## Task 1: To-Do List

Build a task manager application that allows users to add, update, delete, and view daily tasks. The goal of this project is to practice list handling, menu-driven programming, and basic CRUD operations in Python.

![Task Manager Demo](Sources/TaskManger.gif)

## Task 2: Calculator

Create a simple calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division based on user input. This task helps practice operators, conditions, and function-based logic.

![Calculator Demo](Sources/Task-2.gif)


## Task 3: Password Generator

Develop a password generator that creates strong and random passwords using letters, numbers, and special characters. This project is useful for practicing randomization and string handling in Python.

![Password Generator Demo](Sources/Task-3.gif)


## Task 4: Rock Paper Scissors Game

Build a Rock, Paper, Scissors game where the user plays against the computer. This task is designed to strengthen concepts like condition checking, random choices, and interactive game flow.

![Rock Paper Scissors Demo](Sources/Task-4.gif)


## Task 5: Contact Book

Create a contact management system that allows users to add, view, search, update, and delete contacts. This project strengthens your understanding of data structures (lists and dictionaries), menu-driven programming, and CRUD operations in Python.

![Contact Book Demo](Sources/Task-5.gif)
